# Stock Market Analyzer (NumPy Project)

## 📌 Overview
This project is a simple stock market data analysis tool built using Python and NumPy.  
It analyzes stock price data and generates insights like growth, volatility, and performance comparison.

---

## 📊 Dataset
The dataset is a NumPy 2D array where:

- Each row represents a different stock
- Each column represents daily closing prices

Example:

```python
stocks = np.array([
    [100,102,101,105,110,108,115],
    [50,55,53,60,58,62,65],
    [200,198,205,210,215,220,218]
])
